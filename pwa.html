Progressive Web App

I'm also creating this website to demo a new breed of web applications, called Progressive Web Apps or PWA from Google. Since these features are in beta phase expect minor inconsistency. There are many great features, click to show details:

Offline mode
i.e. the pages already visited are available through the progressive web app at all times, even without internet. Try it now, put the phone on airplane mode and visit this page. Also notice that unlike other websites instead of getting an error screen, it is possible to show a custom offline page. By allowing visitors offline access, businesses can potentially increase their customer retention and engagement rates dramatically in countries where users pay for the internet by data usage, allowing people to browse in offline mode can be an extra incentive for visitors to choose the business with a progressive web app, over one that doesn’t.

Website which is also a mobile app
PWA’s are often designed to look and feel like mobile applications, letting users operate in familiar settings while still having the full functionality of websites. You can install this app on mobile devices without developing a separated Android or iPhone application. More and more browsers are automatically prompting users to install the progressive web app for them. If you got the prompt try adding this site as an app to your phone or go here to install it on your iPhone, while Native Support for iOS is in development. Customized push notification allows business to keep in touch with customers on the go. As demo, after installing this website you will get one motivational notification daily.

Once you've subscribed your user, you'd send their subscription to your server to store in a database so that when you want to send a message you can lookup the subscription and send a message to it.

Improved performance
As you may have already noticed, Progressive Web Apps work significantly faster(>70% faster and smoother) due to the way the underlying technology caches and serve text, stylesheets, images and other content on the websites.
